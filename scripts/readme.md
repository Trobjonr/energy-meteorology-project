# Information

.py scripts were created to be run on Python 3.10 or above. The following prerequisite libraries are required for several, if not all, of the scripts:

pandas 

numpy 

matplotlib 

I run my scripts on Anaconda (conda).

# Scripts

VD_#_lateral.py : scripts that output velocity deficit graphs.

combined_visualization.py : visualization of flight path and velocity, overlaid on elevation map.

data_combiner.py : compiles sonic anemometer data and outputs stability plots.

elevation_and_zones.py : visualization of flight path split into zones, overlaid on elevation map.

elevation_plot.py : plots elevation map for reference.

lidar_comparison2.py : compares LIDAR data to aircraft data for statistical validation.

speed_TKE_visualization.py : produces median wind speed and TKE plots

terrain_analysis.py : plots elevation map with median wind speed and TKE plots.

turbine_data.py : compiles turbine locations into .csv files

upstream_vs_wake.py : plots upstream vs wake zones.
